# Fr.Kyrollos-Books-App

## About the app
  This is android app that displays religious books and enable user to download any book for free  
***

## Technologies Used
  * Kotlin Coroutines
  * Dagger Hilt
  * Firebase Firestore and Firebase Storage
  * MVVM Pattern
  * Solid Principles
